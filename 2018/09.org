* Part One

See the code. De spelregels zijn daar kort uitgeschreven voor elke code statement.
Wat je hier moet realiseren is dat 'last marble is worth =x= points' betekend 'er zijn =x= marbles'.

Moest nog wel ff denken over hoe je bijhoud welke speler aan de beurt is maar dat is natuurlijk =marble % num_players= omdat je elke marble op volgorde speelt.

Het bijhouden van de scores dacht ik eerst een default dict. Maar het
initializeren van een fixed array met 0-en is net zo gemakkelijk. ~scores = [0] *
num_players~
